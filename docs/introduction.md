# Introduction 👋

Welcome to **Collaborative Docs** — a team-maintained documentation project
built using GitHub workflow and Docsify.

## What is this project?
This platform allows a team of contributors to collaboratively write and
maintain documentation using:
- **GitHub Issues** → Track what needs to be documented
- **Branches** → Isolate each contributor's work
- **Pull Requests** → Submit changes for review
- **Merging** → Accepting approved changes

## Team Roles
| Role | Responsibility |
|------|---------------|
| Maintainer | Reviews PRs, merges changes |
| Contributor | Creates issues, writes docs, submits PRs |
