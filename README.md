# 📄 Collaborative Docs

A collaboratively maintained documentation project built using GitHub workflow and Docsify.

## 🌐 Live Site
> Coming soon via GitHub Pages

## 📁 Project Structure
- `docs/` → All documentation files
- `index.html` → Docsify website entry point
- `CONTRIBUTING.md` → How to contribute
- `.github/ISSUE_TEMPLATE/` → Issue templates

## 👥 How to Contribute
1. Open an **Issue** describing your change
2. Create a **branch** and make your edits
3. Submit a **Pull Request**
4. Wait for **review and merge**

## 🚀 Run Locally
```bash
npx serve .
```
Then open `http://localhost:3000`
